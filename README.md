# ZN'mood - Check Your Mental Health

---

## 🇮🇩 Bahasa Indonesia

### Deskripsi Proyek
**ZN'mood** adalah sebuah website sederhana yang dirancang sebagai proyek pertama saya dalam mempelajari pemrograman web (HTML, CSS, dan JavaScript). Website ini dibuat sebagai alat bantu interaktif untuk meningkatkan kesadaran dan membantu pengguna mengecek kondisi kesehatan mental mereka secara mandiri.

### Fitur Utama
1. **Kuis Tingkat Depresi:** Pengguna dapat menjawab beberapa pertanyaan/kuis seputar kondisi mental mereka. Setiap pilihan jawaban memiliki bobot poin tersendiri yang akan dihitung secara otomatis di akhir kuis untuk menentukan estimasi tingkat depresi pengguna.
2. **Konsultasi (Prototipe):** Fitur ini disediakan sebagai jembatan informasi untuk pengguna yang membutuhkan bantuan lebih lanjut. *Catatan: Fitur ini belum terhubung ke sistem backend/pesan karena web ini masih bersifat halaman statis sederhana.*
3. **Rekomendasi Dokter:** Halaman ini menyediakan informasi dan daftar beberapa rekomendasi dokter spesialis kedokteran jiwa (psikiater) atau psikolog yang dapat dihubungi oleh pengguna untuk penanganan profesional.

## 🇬🇧 English

### Project Description
**ZN'mood** is a simple website developed as my very first project to learn web development (HTML, CSS, and JavaScript). This website serves as an interactive tool to raise awareness and help users perform a self-check on their mental health condition.

### Key Features
1. **Depression Level Quiz:** Users can answer a series of questions regarding their mental state. Each answer selection has specific point values that will be automatically calculated at the end of the quiz to estimate the user's depression level.
2. **Consultation (Prototype):** This feature is intended to guide users who need further assistance. *Note: This feature is currently not connected to any backend or live messaging system as this is still a simple static website.*
3. **Doctor Recommendations:** This section provides information and a list of recommended psychiatrists or psychologists that users can reach out to for professional help.
